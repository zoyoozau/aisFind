# 📋 คู่มือการติดตั้งและ Deploy ระบบ aisFix

โปรเจกต์ **aisFix** แบ่งโครงสร้างการทำงานเป็น 2 ส่วนหลัก คือ:
1. **Backend:** รันอยู่บน Google Sheets ผ่าน **Google Apps Script** ทำหน้าที่เป็นฐานข้อมูลบันทึกข้อมูล
2. **Frontend:** หน้าเว็บระบบ SPA สไตล์ Premium Dark Mode ที่ผู้ใช้ล็อกอินและป้อนเลข 10 หลัก สามารถรันบนเครื่องของคุณ หรืออัปโหลดขึ้น **Vercel** เพื่อเปิดใช้งานผ่านโทรศัพท์มือถือได้อย่างสะดวกสบาย

---

## 🛠️ ขั้นตอนที่ 1 — ตั้งค่า Google Sheet และ Apps Script (Backend)

1. **สร้างหรือเปิด Google Sheet** ที่ต้องการใช้ในการจัดเก็บข้อมูล
2. ไปที่เมนู **Extensions (ส่วนขยาย) → Apps Script**
3. **ลบ** โค้ดเริ่มต้นเดิมในไฟล์ `Code.gs` ทั้งหมด
4. **คัดลอกโค้ด** จากไฟล์ `apps-script-code.gs` ในโปรเจกต์นี้มาวางลงไปทั้งหมด
5. กดปุ่ม **Save (บันทึก)** หรือกด `Ctrl + S` (`Cmd + S` บน Mac)
6. ทำการติดตั้งเพื่อเปิดบริการเว็บ (Deploy):
   * คลิกที่ปุ่ม **Deploy (การทำงานใช้จริง) → New deployment (การทำงานใช้จริงใหม่)**
   * เลือกประเภทของ Deployment เป็น **Web app (เว็บแอป)** (คลิกรูปฟันเฟืองเพื่อเลือกหากไม่ขึ้นรูป)
   * ตั้งค่าข้อมูลดังนี้:
     * **Description:** aisFix API
     * **Execute as (เรียกใช้ในฐานะ):** **Me (บัญชีอีเมลของคุณเอง)**
     * **Who has access (ผู้มีสิทธิ์เข้าถึง):** **Anyone (ทุกคน)** *(สำคัญมาก! หากไม่เลือก Anyone ตัวหน้าเว็บจะเข้าถึงชีตไม่ได้)*
   * คลิกปุ่ม **Deploy**
   * หากมีปุ่มเด้งขึ้นมา ให้เลือก **Authorize access (ให้สิทธิ์การเข้าถึง)** → คลิกเลือกบัญชี Google ของคุณ → คลิก **Advanced (ขั้นสูง)** ที่มุมซ้ายล่าง → เลือก **Go to Untitled project (unsafe)** หรือ **ไปยังโปรเจกต์ (ไม่ปลอดภัย)** → กด **Allow (อนุญาต)**
7. เมื่อเสร็จสิ้น คุณจะได้ **Web app URL** (จะขึ้นต้นด้วย `https://script.google.com/macros/s/.../exec`) 
   * **คัดลอก (Copy) URL นี้เก็บไว้** เพื่อนำไปใช้เชื่อมโยงกับหน้าเว็บในขั้นตอนถัดไป

> [!NOTE]
> * ตัวสคริปต์จะทำการสร้างแผ่นงาน (Sheet) ใหม่ชื่อ **`Users`** และ **`Data`** ให้โดยอัตโนมัติเมื่อมีการทดสอบหรือใช้งานครั้งแรก
> * หากมีการแก้ไขหรืออัปเดตโค้ดเบื้องหลังใน Apps Script ในภายหลัง คุณต้องเลือก **Deploy → Manage deployments → Edit (รูปดินสอ) → เลือก Version เป็น New Version** แล้วกด **Deploy** ทุกครั้งเพื่อให้โค้ดเวอร์ชันล่าสุดทำงาน

## 🚀 ขั้นตอนที่ 2 — วิธี Deploy หน้าเว็บขึ้น Vercel (Frontend)

คุณสามารถเปิดไฟล์ `index.html` บนบราวเซอร์ในเครื่องคอมพิวเตอร์ของคุณโดยตรงเพื่อใช้งานได้ทันที แต่หากต้องการเปิดใช้งานผ่านหน้าเว็บออนไลน์หรือบนสมาร์ทโฟน แนะนำให้ Deploy ขึ้น Vercel ซึ่งมีขั้นตอนที่ทำตามได้ง่ายมากๆ:

### วิธีที่ 1: ลากและวางโฟลเดอร์ (ง่ายที่สุด)
1. เข้าสู่ระบบหรือลงทะเบียนที่ [vercel.com](https://vercel.com)
2. เมื่อเข้าสู่หน้าแดชบอร์ดหลัก ให้กดปุ่ม **Add New... → Project**
3. ด้านล่างของตัวเลือก จะมีส่วนสำหรับอัปโหลด ให้เลือกอัปโหลดโฟลเดอร์ `aisFix` ที่มีไฟล์ `index.html` และ `vercel.json` อยู่
4. กดปุ่ม **Deploy**
5. รอระบบประมวลผลประมาณ 1 นาที → คุณจะได้ลิงก์หน้าเว็บออนไลน์สำหรับเข้าใช้งานทันที! 📱

---

## 🔗 ขั้นตอนที่ 3 — วิธีเข้าใช้งานและเชื่อมต่อระบบครั้งแรก

1. เปิดหน้าเว็บที่คุณ Deploy บน Vercel หรือดับเบิลคลิกเปิดไฟล์ `index.html` ในคอมพิวเตอร์ของคุณ
2. ระบบจะพบว่ายังไม่มีการเชื่อมต่อ API และแสดงแถบแจ้งเตือนสีเหลืองด้านบนว่า *"ระบบยังไม่ได้เชื่อมต่อ Google Sheets API..."*
3. คลิกที่ปุ่ม **"ตั้งค่าเดี๋ยวนี้"** หรือคลิกที่ไอคอน **ฟันเฟือง (Settings)**
4. วาง **Google Apps Script Web App URL** ที่ได้จากขั้นตอนที่ 1 ลงในช่องกรอกข้อมูล
5. คลิกปุ่ม **"ทดสอบการเชื่อมต่อ"** เพื่อเช็กว่าระบบสามารถคุยกับ Google Sheet ได้อย่างถูกต้องหรือไม่
6. เมื่อการเชื่อมต่อสำเร็จเรียบร้อย ให้กดปุ่ม **"บันทึกการตั้งค่า"** (ระบบจะจำค่านี้ไว้ใน LocalStorage บนเบราว์เซอร์นั้นๆ โดยอัตโนมัติ ทำให้คุณไม่ต้องกรอกซ้ำอีกในครั้งถัดไป)
7. แถบแจ้งเตือนสีเหลืองจะหายไป และระบบพร้อมล็อกอินใช้งานทันที!

## 🔑 บัญชีเข้าสู่ระบบเริ่มต้น (Default Accounts)

ตัวสคริปต์ Backend จะเพิ่มบัญชีตัวอย่างให้คุณอัตโนมัติในชีต `Users` เมื่อสร้างระบบครั้งแรก ซึ่งคุณสามารถนำมาล็อกอินหน้าเว็บได้ทันที:

| สิทธิ์ผู้ใช้งาน | ชื่อผู้ใช้ (Username) | รหัสผ่าน (Password) |
| :--- | :--- | :--- |
| **ผู้ดูแลระบบ (Admin)** | `admin` | `1234` |
| **ผู้ใช้ทั่วไป 1** | `user1` | `1234` |
| **ผู้ใช้ทั่วไป 2** | `user2` | `1234` |

> [!TIP]
> คุณสามารถเปิด Google Sheets เพื่อเข้าไปลบ เพิ่มเติม หรือปรับเปลี่ยน Username / Password ของแต่ละคนได้โดยตรงที่แท็บชีต **`Users`** ได้ตลอดเวลา!
